Tries to load a valid VRTK_SDKSetup from a list.

The first loadable VRTK_SDKSetup in the list will be loaded. Will fall back to disable VR if none of the provided Setups is useable.

Parameters
startIndexThe index of the VRTK_SDKSetup to start the loading with.
tryToReinitializeWhether or not to retry initializing and using the currently set but unusable VR Device.
sdkSetupsThe list to try to load a VRTK_SDKSetup from.
