Detailed Description

The Player Climb allows player movement based on grabbing of VRTK_InteractableObject objects that have a Climbable grab attach script.

Because it works by grabbing, each controller should have a VRTK_InteractGrab and VRTK_InteractTouch component attached.

VRTK/Examples/037_CameraRig_ClimbingFalling shows how to set up a scene with player climbing. There are many different examples showing how the same system can be used in unique ways.
