◆ calculateDragForce()

Vector3 FreeFallBehavior.calculateDragForce ( )
private

Calculates the drag Force of the Player.

A private function, this is only called within applyDragForce. Its lack of parameters is unlikely to change because all the values used in the calculation are physics-based traits of the player and the world. Likewise, its output is the same type as the parameter taken by the addForce method from the external library and will certainly be stable.

Returns
Returns a Vector3 that represents the Drag Force Vector

void FreeFallBehavior.updateAngularDrag ( )

Experimental angular drag setting based on proximity of hands.

Parameters are unnecessary due to our ability to obtain the necessary information as class variables or library calls, and our setting state through another library call. Holding hands in increases angular drag and allows a quick slowing of rotation
