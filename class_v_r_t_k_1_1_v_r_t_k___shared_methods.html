static T VRTK.VRTK_SharedMethods.FindEvenInactiveComponent< T > ( )
static

Finds the first component of a given type.

This method returns components from active as well as inactive GameObjects in the scene. It doesn't return assets. For performance reasons it is recommended to not use this function every frame. Cache the result in a member variable at startup instead.

Template Parameters
TThe component type to search for. Must be a subclass of Component.
Returns
The found component. If no component is found is returned.
Type Constraints
T :Component

◆ FindEvenInactiveGameObject< T >()

static GameObject VRTK.VRTK_SharedMethods.FindEvenInactiveGameObject< T > ( string  gameObjectName = null)
static

Finds the first GameObject with a given name and an ancestor that has a specific component.

This method returns active as well as inactive GameObjects in the scene. It doesn't return assets. For performance reasons it is recommended to not use this function every frame. Cache the result in a member variable at startup instead.

Template Parameters
TThe component type that needs to be on an ancestor of the wanted GameObject. Must be a subclass of Component.
Parameters
gameObjectNameThe name of the wanted GameObject. If it contains a '/' character, this method traverses the hierarchy like a path name, beginning on the game object that has a component of type T .
Returns
The GameObject with name gameObjectName and an ancestor that has a T . If no such GameObject is found is returned.
Type Constraints
T :Component

The SetOpacity method allows the opacity of the given GameObject to be changed.
