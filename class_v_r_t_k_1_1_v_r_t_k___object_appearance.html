Detailed Description

The Object Appearance script provides a collection of static methods for controlling the appearance of a GameObject.

The GameObject can have it's opacity changed, or it's renderers toggled, or highlighters toggled.

static

The SetOpacity method allows the opacity of the given GameObject to be changed.

A lower alpha value will make the object more transparent, such as 0.5f will make the controller partially transparent where as 0f will make the controller completely transparent.

Parameters
modelThe GameObject to change the renderer opacity on.
alphaThe alpha level to apply to opacity of the controller object. 0f to 1f.
transitionDurationThe time to transition from the current opacity to the new opacity.
