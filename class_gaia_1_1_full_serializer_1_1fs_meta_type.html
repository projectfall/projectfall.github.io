◆ CreateInstance()

object Gaia.FullSerializer.fsMetaType.CreateInstance ( )

Creates a new instance of the type that this metadata points back to.

If this type has a default constructor, then Activator.CreateInstance will be used to construct the type (or Array.CreateInstance if it an array). Otherwise, an uninitialized object created via FormatterServices.GetSafeUninitializedObject is used to construct the instance.
