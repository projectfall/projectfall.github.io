Detailed Description

This canvas adds the unity console log to a world game object.

To use the prefab, it simply needs to be placed into the scene and it will be visible in world space. It's also possible to child it to other objects such as the controller so it can track where the user is.

It's also recommended to use the Simple Pointer and UI Pointer on a controller to interact with the Console Viewer Canvas as it has a scrollable text area, a button to clear the log and a checkbox to toggle whether the log messages are collapsed.

void VRTK.VRTK_ConsoleViewer.SetCollapse ( bool  state)

The SetCollapse method determines whether the console will collapse same message output into the same line.

A state of true will collapse messages and false will print the same message for each line.

Parameters
stateThe state of whether to collapse the output messages, true will collapse and false will not collapse.
