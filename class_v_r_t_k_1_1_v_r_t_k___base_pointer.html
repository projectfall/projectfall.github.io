Detailed Description

This abstract class provides any game pointer the ability to know the state of the implemented pointer.

It extends the VRTK_DestinationMarker to allow for destination events to be emitted when the pointer cursor collides with objects.

As this is an abstract class, it cannot be applied directly to a game object and performs no logic.

States of Pointer Visibility.

Parameters
On_When_ActiveOnly shows the pointer beam when the Pointer button on the controller is pressed.
Always_OnEnsures the pointer beam is always visible but pressing the Pointer button on the controller initiates the destination set event.
Always_OffEnsures the pointer beam is never visible but the destination point is still set and pressing the Pointer button on the controller still initiates the destination set event.
