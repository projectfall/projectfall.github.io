Detailed Description

All 3D controls extend the VRTK_Control abstract class which provides a default set of methods and events that all of the subsequent controls expose.

◆ SetContent()

virtual void VRTK.VRTK_Control.SetContent ( GameObject  content,
bool  hideContent 
)
virtual

The SetContent method sets the given game object as the content of the control.

This will then disable and optionally hide the content when a control is obscuring its view to prevent interacting with content within a control.

Parameters
contentThe content to be considered within the control.
hideContentWhen true the content will be hidden in addition to being non-interactable in case the control is fully closed.
