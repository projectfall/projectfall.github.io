◆ SpawnerRuleSelector

enum Gaia.GaiaConstants.SpawnerRuleSelector
strong

The algorithm which defines which rule will be selected in spawn location.

Regardless of whether selected or not, the rule will still only run if it exceeds its minimum fitness settings, and didnt randomly fail (failure rate). All - All rules will be run at this location Fittest - Only the fittest rule will be run at this location WeightedFittest - The fittest rule will most likely run, then the next fittest, distributed by relative fitness Random - A random rule will be chosen and evaluated to run

Fittest
