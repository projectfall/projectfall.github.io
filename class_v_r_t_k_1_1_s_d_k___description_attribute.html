Detailed Description

Describes a class that represents an SDK.

Only allowed on classes that inherit from SDK_Base.

Creates a new attribute.

Parameters
prettyNameThe pretty name of the SDK. Uniquely identifies the SDK. and string.Empty aren't allowed.
symbolThe scripting define symbol needed for the SDK. Needs to be the same as SDK_ScriptingDefineSymbolPredicateAttribute.symbol to add and remove the scripting define symbol automatically using VRTK_SDKManager. and string.Empty are allowed.
vrDeviceNameThe name of the VR Device to load. Set to or string.Empty if no VR Device is needed.
buildTargetGroupNameThe name of a constant of BuildTargetGroup. "<see cref="BuildTargetGroup.Unknown"/>", and string.Empty are not allowed.
indexThe index of this attribute, in case there are multiple on the same target.
