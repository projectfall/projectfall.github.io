Detailed Description

This class implements a Django-like simple templating system for creating clean dynamic html or other text without using asp.net server controls or string builders that require html and other text to be written and escaped in .cs files.

The template is very high performance, since it indexes a template once (lazy loading the first time) and uses a string builder insert after that to keep memory use and copy time to a minimum.

Example text file: "Hey {{ Name }}, what's up?" Example Template.ToString() output after calling Set("Name", "Craig"); : "Hey Craig, what's up?"
