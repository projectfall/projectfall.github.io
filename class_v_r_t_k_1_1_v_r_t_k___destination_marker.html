Detailed Description

This abstract class provides the ability to emit events of destination markers within the game world.

It can be useful for tagging locations for specific purposes such as teleporting.

It is utilised by the VRTK_BasePointer for dealing with pointer events when the pointer cursor touches areas within the game world.
